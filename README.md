# Horror: Awakening / Хоррор: Пробуждение

**3D хоррор на Unity** — шутер от первого лица + выживание + открытый мир.

## 🎮 Описание

Вы просыпаетесь в заброшенном городке. Психиатрическая больница, церковь, морг, кладбище, лес — всё окутано тьмой. Монстры бродят по улицам. В подземном бункере ждёт БОСС. Задача: выжить, найти оружие и уничтожить источник зла.

## 🎯 Что в игре

| Что | Подробнее |
|-----|-----------|
| 🔫 Оружие | Пистолет + дробовик, перезарядка, вспышка |
| 👻 Монстры (4 типа) | Тень / Крикун (телепорт) / Ведьмак / Босс с 3 фазами |
| 🗺️ Открытый мир | 200×200 procedural town: здания, дороги, лес, кладбище |
| 🏚️ Здания | Можно заходить, исследовать комнаты |
| 💊 Предметы | Аптечки, патроны, ключи, батарейки, адреналин |
| 🧠 Рассудок | Падает от монстров/темноты. Ноль = смерть |
| 🔦 Фонарик | Батарея садится |
| 📱 Мобильное | Джойстик слева, поворот справа, кнопки |
| 💾 Сохранения | JSON-файл |
| 🎬 Эффекты | Тряска камеры, виньетка, туман, частицы |

## 🏗️ Как собрать APK

### Установить:
1. [Unity Hub](https://unity.com/download) → установи **Unity 2022.3 LTS**
2. При установке выбери модуль **Android Build Support**
3. JDK 17 + Android SDK (ставится с Unity)

### Сборка:
1. `git clone https://github.com/Sokp6/Horror-.git`
2. Открой папку в Unity Hub → дождись импорта
3. `File → Build Settings → Android → Switch Platform → Build`
4. APK будет в `Builds/Android/`

### Быстрая сборка:
В Unity Editor: меню `Horror Game → Build Android APK`

## 🎮 Управление

**ПК:** WASD / Мышь / ЛКМ / R / Shift / C / Пробел
**Телефон:** Левая половина — джойстик / Правая — поворот / Кнопки — огонь, перезарядка

## 📁 Код

```
Assets/Scripts/
├── Player/         FPSController.cs — ходьба, камера, присед, бег
├── Weapons/        WeaponBase.cs — пистолет/дробовик, перезарядка, отдача
├── Monsters/       MonsterAI.cs — патруль/погоня/атака, 4 типа
│                   BossAI.cs — босс с фазами, призыв миньонов, снаряды
├── World/          WorldGenerator.cs — procedural open world
│                   ItemPickup.cs — собираемые предметы, двери
├── UI/             HUDController.cs — здоровье/рассудок/патроны
│                   MobileInput.cs — тач-управление
├── Systems/        GameManager.cs — синглтон, состояние игры
│                   SaveSystem.cs — JSON сохранения
└── Editor/         BuildScript.cs — сборка APK в 1 клик
```
