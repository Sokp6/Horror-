# Horror: Awakening / Хоррор: Пробуждение

Текстовый хоррор-квест для Android на Kotlin. Действие происходит в заброшенной советской психиатрической больнице.

## 📖 Сюжет

Вы просыпаетесь в заброшенной психиатрической лечебнице. Вы не помните ни своего имени, ни того, как здесь оказались. Вокруг — полуразрушенные коридоры, тёмные палаты и... что-то в подвале, что ждёт вас.

Исследуйте 11 комнат, собирайте предметы, раскрывайте тайну «Нулевого пациента» и постарайтесь выбраться живым.

## 🎮 Геймплей

- **Исследование**: Перемещайтесь между комнатами, осматривайте окружение
- **Инвентарь**: Собирайте и используйте предметы
- **Система здоровья и рассудка**: Следите за обоими показателями
- **Множественные концовки**: 4 разные концовки (хорошая, плохая, секретная, безумие)
- **Скримеры**: Визуальные и вибрационные эффекты
- **Сохранение**: Автоматическое сохранение при выходе

## 🎯 Концовки

| Концовка | Как получить |
|----------|-------------|
| **Освобождение** (хорошая) | Спуститесь в подвал с амулетом и страницей заклинания, используйте револьвер |
| **Поглощение** (плохая) | Спуститесь в подвал без подготовки и выстрелите |
| **Принятие** (секретная) | Найдите дневник, историю болезни и медкарту; в подвале НЕ стреляйте |
| **Безумие** (безумие) | Дайте рассудку упасть до -20 |

## 🏗️ Сборка

Откройте проект в **Android Studio** (Hedgehog 2023.1.1+).

### Требования

- Android Studio Hedgehog или новее
- JDK 17
- Android SDK 34
- Gradle 8.5 (встроен в wrapper)

### Сборка APK

```bash
# Debug APK
./gradlew assembleDebug

# Release APK  
./gradlew assembleRelease
```

APK будет находиться в `app/build/outputs/apk/`.

## 🗺️ Структура проекта

```
app/src/main/java/com/horrorgame/awakening/
├── MainActivity.kt          # Главное меню
├── GameActivity.kt          # Игровой экран
└── game/
    ├── GameEngine.kt        # Игровой движок
    ├── StoryData.kt         # Сюжет, комнаты, предметы
    ├── Player.kt            # Состояние игрока
    ├── Room.kt              # Модель комнаты
    ├── Item.kt              # Модель предмета
    ├── SoundManager.kt      # Звук и вибрация
    └── SaveManager.kt       # Сохранения
```
